# UCI---Autism-Spectrum-Disorder-data-of-Children# UCI Autism Spectrum Disorder Data of Children

## Overview

This project analyzes the UCI Autism Spectrum Disorder Screening Data for Children using Python in Google Colab. The notebook explores the dataset, performs data preprocessing, visualization and applies machine learning techniques to predict whether a child is likely to have Autism Spectrum Disorder based on screening questionnaire responses and demographic information.

## Dataset

- UCI Autism Spectrum Disorder Screening Data for Children
- UCI Machine Learning Repository

## Objectives

- Explore and understand the dataset
- Clean and preprocess the data
- Perform Exploratory Data Analysis (EDA)
- Visualize important features
- Train and evaluate machine learning models
- Predict ASD screening outcomes

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Features

- Data preprocessing
- Missing value handling
- Exploratory Data Analysis (EDA)
- Data visualization

- Model evaluation
